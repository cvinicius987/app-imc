## Aplicação para Cálculo do IMC

Aplicação que realiza o cálculo do IMC (Indice de Massa Corpóral).

### Stack
- Kotlin
- Maven
- Docker
- Spring Boot

### Uso
Após o clone execute o seguinte comando:
```
mvn spring-boot:run

ou 

./mvnw spring-boot:run
```

A aplicação irá funcionar em http://localhost:8080
